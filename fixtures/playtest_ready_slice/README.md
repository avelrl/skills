# Courier Drift

Playtest-ready fixture for a first playable web slice.

## Current State

- the project already has a Vite + TypeScript + Canvas scaffold
- `Player Movement` and `Combat Loop` are integrated into one runnable MVP scene
- the slice has boot, play, win, loss, and restart flow
- `reports/mvp-assembly-report.md` exists; `reports/playtest-report.md` does not

## What Needs Validation

- combat readability and telegraph clarity
- timer pressure, pacing, and HUD comprehension
- small feel fixes that improve the current loop without adding new systems

## Run

2. `npm run dev`
