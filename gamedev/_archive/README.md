# Gamedev Archive

This directory keeps good but non-core gamedev material out of the active path.

Use it for:
- management or release-process files that are not part of the default workflow,
- worldbuilding and content-production templates,
- domain standards that are useful only for specific projects.

Default rule:
- if an active skill or active template depends on a file, it does not belong here.
- if a file is useful but not part of the default `setup-engine -> map-systems -> design-system -> prototype` path, archive it here instead of deleting it.
